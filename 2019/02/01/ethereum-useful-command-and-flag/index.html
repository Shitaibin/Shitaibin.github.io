以太坊有用的命令和标记

发表于 2019-02-01 更新于 2020-12-13 阅读次数：

--debug：日志显示文件和行数

--pprof：节点开启http服务用来显示prof信息，网址和端口可以通过--pprofaddr, --pprofport指定，默认是127.0.0.1:6060。

打开网址：http://127.0.0.1:6060/debug/pprof/

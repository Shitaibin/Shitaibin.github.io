SSH常用命令和配置

发表于 2020-07-28 更新于 2020-12-13 阅读次数：

生成SSH密钥

-t：指定加密算法
-f：指定路径
-C：注释，可以填写用户名或邮箱
-N：密码

指定以上f、C、N这3个参数，可以避免交互式问答，快速生成密钥，在脚本中使用很方便。

SSH客户端配置文件

~/.ssh/config文件内容如下：

  • Host：自定义命名
  • HostName：机器IP或者域名
  • User：登录机器的用户名
  • Port：登录机器的端口，默认为22，可省略
  • IdentityFile：登录机器时使用的私钥，默认为~/.ssh/id_rsa，可省略；当某台机器使用单独密钥时，很有用
